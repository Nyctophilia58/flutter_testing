# Flutter Testing 🧪

This repository contains sample **unit tests** and **widget tests** for Flutter projects.  
It is intended as a reference or learning resource for practicing different types of testing in Flutter apps.


## 📂 Project Structure

The repository is organized into separate folders for different testing approaches:

- **`unit_test/`** – Contains examples of basic unit tests  
- **`widget_test/`** – Contains examples of widget tests  

Each folder is an independent Flutter project with its own `pubspec.yaml`.


## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/flutter_testing.git
   cd flutter_testing
   
2. Navigate into a project folder:
   ```bash
   cd unit_test
   ```
   or
   ```bash
   cd widget_test

3. Get dependencies:
   ```bash
   flutter pub get

4. Run tests:
   ```bash
   flutter test

## 🧩 What’s Included

- Unit Tests
  - Testing pure Dart classes and methods
  - Verifying logic without UI

- Widget Tests
  - Testing UI components in isolation
  - Verifying widgets’ behavior and rendering
